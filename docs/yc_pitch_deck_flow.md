# YC Pitch Deck Flow: Phantom

## The Narrative Arc
This deck is designed for a 3-minute YC pitch or a rapid investor screen. It leans into the "Tech-Bro / Founder" pain points: fast shipping, PMF velocity, and the dread of beta testing.

---

### Slide 1: Title & Hook
**Visual:** Minimalist dark mode. The Phantom logo.
**Header:** Phantom
**Subhead:** The Infrastructure for PMF Velocity.
**Talk Track:** "We are Phantom. We simulate your exact target audience's cognition at scale, so you can iterate 100x faster."

### Slide 2: The Core Problem (The Death Spiral)
**Visual:** A descending graph showing 'Time to Value' vs 'Conversion Rate'.
**Header:** The Feedback Loop is Broken.
**Bulleted Points:**
- The #1 reason startups die is building the wrong thing.
- Human beta testers (B2B & B2C) are too slow and expensive to recruit.
- Puppeteer load scripts only check if the server crashes, not if the UI is confusing.
**Talk Track:** "Whether you are building enterprise SaaS or consumer social, getting your target group to test an MVP is a massive blocker. Founders can't wait 3 weeks. So they launch blind, burn their early adopters on friction-heavy flows, and die."

### Slide 3: The Solution (Synthetic Reality)
**Visual:** Diagram showing Code -> Phantom CI/CD Stage -> "Ghosts" Swarming -> Friction Report -> Analytics.
**Header:** The Infinite Cohort.
**Bulleted Points:**
- Autonomous Visual LLM Agents.
- Calibrated Human Personas (e.g., "Distracted B2B Buyer", "Impatient Gen-Z").
- Plugs directly into GitHub Actions / CI/CD.
**Talk Track:** "Phantom is the missing cognitive data layer. You hook us into your CI/CD, and we spin up 50 autonomous visual agents loaded with the psychological traits of your target group. They navigate your staging app and find exact cognitive bottlenecks."

### Slide 4: The "Unfair Advantage" / Technology Moat
**Visual:** Split screen. Left: Vague Jira ticket ("This looks weird"). Right: Phantom Output (DOM trace, Screenshot, Scan-Path Entropy score).

**Talk Track:** "We aren't just sending dumb bots. Our agents evaluate human heuristics. We turn a $10k focus group into a 3-minute automated pipeline, giving you deterministic data on where a real human would rage-quit."
