# Phantom Brand Kit v1.0

> **The Visual Substrate for Synthetic Reality.**

Phantom is an infrastructure tool. Our brand must convey **Transparency**, **Observed Intelligence**, and **Industrial Reliability**.

## 🎨 Color Palette (The spectral Spectrum)

| Name | Hex | HSL | Use Case |
| :--- | :--- | :--- | :--- |
| **Obsidian Glass** | `#0a0a0c` | `240, 14%, 4%` | Base Surface, Deep UI |
| **Phantom Orange** | `#ea580c` | `20, 90%, 48%` | Primary Action, Critical Friction |
| **Ethereal Purple** | `#8a2be2` | `271, 76%, 53%` | Cognition, LLM Inference, AI State |
| **Friction Red** | `#ff7b72` | `3, 100%, 72%` | Errors, Chaos Mode, High Resistance |
| **Success Cyan** | `#3fb950` | `131, 49%, 49%` | Value Proposition, "Aha! Moments" |

## 📐 Typography

- **Primary Headings**: `Inter` (Bold, Tracking +0.05em)
- **Technical/Mono**: `JetBrains Mono` or `Roboto Mono`
- **Voice**: Precise, cold, yet authoritative.

## 🖼 Brand Assets

1. **The Phantom Mark**: The original high-fidelity geometric logo used in production.
2. **Standard Typography**: Inter (Bold) for headings, simple and authoritative.

---

## 🚀 Usage Guidelines

- **Onboarding**: Use Phantom Orange for CTAs to drive high-intent calibration.
- **Reporting**: Transition to Success Cyan when PMF is detected.
- **Roast Mode**: Shift UI borders to Friction Red to emphasize critical failure.
