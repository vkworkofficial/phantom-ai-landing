# Deferred Features (S26 Post-Purge)

The following features were removed from the immediate production build to achieve stability and a lean product focus as of March 2026. These are documented for re-integration once the core platform is stable.

## Marketing & SEO
- **Blog**: Full blog infrastructure and content removed.
- **Programmatic SEO (pSEO)**: Automated landing page generation strips and templates.

## Advanced Platform Features
- **AI User Testing Page**: Dedicated personas and user testing dashboard.
- **QA Agents**: Automated testing instances previously referred to as "Verification Agents".
- **PMF Tracking (Forensics)**: Deep-dive "Rage-click" and "Friction Bloom" analysis tools.
- **Integration Hub**: Third-party connection portal.

## Branding Reverts
- **Ghosts / Spirits**: Reverted to "Synthetic Users" or "Instances".
- **Séance / Astral**: Reverted to "Simulation" or "Engine".
- **Forensics / Vibe Audit**: Reverted to "Friction Report" or "Analysis".
